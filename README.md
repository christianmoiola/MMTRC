# MMTRC
Multi-Modal Temporal Relation Classification

---

## Project Setup

### 1. Create a Virtual Environment

```bash
python -m venv venv
```

#### Activate it:

- **Windows**:
  ```bash
  .\venv\Scripts\activate
  ```
- **Linux / macOS**:
  ```bash
  source venv/bin/activate
  ```

### 2. Install NumPy

```bash
pip install numpy
```

### 3. Run the Notebook

Open `dataset_analysis.ipynb` and run it.
